# 3.1.2 Jednoznačnost a víceznačnost

<a id="def-nejednoznacna-gramatika"></a>
!!! Definition "[Definice (Nejednoznačná bezkontextová gramatika)](#def-nejednoznacna-gramatika)"

    ### Nejednoznačná bezkontextová gramatika {#def-nejednoznacna-gramatika}

    **Nejednoznačná (víceznačná) bezkontextová gramatika** je taková bezkontextová gramatika $G$, kde lze pro nějaký řetězec z jazyka $L(G)$ sestavit **alespoň dva různé** derivační stromy.

<a id="def-jednoznacna-gramatika"></a>
!!! Definition "[Definice (Jednoznačná bezkontextová gramatika)](#def-jednoznacna-gramatika)"

    ### Jednoznačná bezkontextová gramatika {#def-jednoznacna-gramatika}

    **Jednoznačná bezkontextová gramatika** je taková bezkontextová gramatika $G$, kde pro všechny řetězce z jazyka $L(G)$ existuje **právě jeden** derivační strom.

<a id="def-nejednoznacny-jazyk"></a>
!!! Definition "[Definice (Nejednoznačný bezkontextový jazyk)](#def-nejednoznacny-jazyk)"

    ### Nejednoznačný bezkontextový jazyk {#def-nejednoznacny-jazyk}

    **Nejednoznačný (víceznačný) bezkontextový jazyk** je takový jazyk, kde **každá** bezkontextová gramatika generující tento jazyk je nejednoznačná (víceznačná).
    
    Neboli neexistuje jednoznačná bezkontextová gramatika, která tento jazyk generuje.

<a id="def-jednoznacny-jazyk"></a>
!!! Definition "[Definice (Jednoznačný bezkontextový jazyk)](#def-jednoznacny-jazyk)"

    ### Jednoznačný bezkontextový jazyk {#def-jednoznacny-jazyk}

    **Jednoznačný bezkontextový jazyk** je takový jazyk, který lze generovat jednoznačnou gramatikou.
