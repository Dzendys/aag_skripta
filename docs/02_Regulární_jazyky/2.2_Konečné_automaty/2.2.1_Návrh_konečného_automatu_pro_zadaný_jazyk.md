# 2.2.1 Návrh konečného automatu pro zadaný jazyk

Při návrhu konečného automatu pro konkrétní jazyk často využíváme specifické konstrukční postupy. Níže jsou uvedeny obecné algoritmy pro úpravu a návrh automatů.

<a id="algo-uplne-urceny"></a>
!!! Algorithm "Doplnění konečného automatu na úplně určený"

    ### Doplnění konečného automatu na úplně určený {#algo-uplne-urceny}

    Pokud máme deterministický automat, který není úplně určený (chybí některé přechody), doplníme jej následovně:

    1. Zavedeme nový **„nulový“ stav** (často značený $q_\emptyset$, *sink state*).
    2. Ke stavu $q_\emptyset$ přidáme smyčku pro všechny symboly vstupní abecedy.
    3. Ze všech stavů, které nemají definovaný přechod pro nějaký symbol vstupní abecedy, přidáme přechod na tento symbol do stavu $q_\emptyset$.

<a id="algo-doplnek"></a>
!!! Algorithm "Konstrukce automatu pro doplněk jazyka"

    ### Konstrukce automatu pro doplněk jazyka {#algo-doplnek}

    Máme automat $M$ přijímající jazyk $L$. Jak získáme automat $M'$ přijímající jazyk $\overline{L}$ (doplněk)?

    1. Automat $M$ musí být **deterministický** a **úplně určený**.
    2. Následně u automatu $M$ **prohodíme koncové a nekoncové stavy** a získáme tak automat $M'$.

<a id="algo-modulo"></a>
!!! Algorithm "Návrh automatu pro dělitelnost binárního čísla"

    ### Návrh automatu pro dělitelnost binárního čísla {#algo-modulo}

    Jak postupujeme při návrhu konečného automatu pro jazyky typu:
    
    $$L = \{w : w \in \{0, 1\}^* \land w \text{ je binární číslo dělitelné číslem } n \text{ se zbytkem } m\}$$

    1. **Počet stavů** automatu bude roven číslu $n$.
    2. Tyto stavy budou reprezentovat **zbytky po dělení** (0, 1, ..., $n-1$).
    3. **Počátečním stavem** bude stav 0.
    4. **Koncovým stavem** bude stav $m$ (neboli stav reprezentující požadovaný zbytek).
    5. Přechodovou funkci sestavíme na základě principu posunu v binární soustavě (připsání 0 zdvojnásobí hodnotu, připsání 1 zdvojnásobí a přičte 1):

    $$
    \begin{aligned}
    \delta(i, 0) &= (2i) \pmod n \\
    \delta(i, 1) &= (2i + 1) \pmod n
    \end{aligned}
    \quad \text{pro } i = 0, 1, \dots, n-1
    $$
