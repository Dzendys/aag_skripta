# 2.3.4 Soustava regulárních rovnic

<a id="algo-soustava-rv"></a>
!!! Algorithm "Řešení soustavy regulárních rovnic"

    ### Řešení soustavy regulárních rovnic {#algo-soustava-rv}

    Jak postupujeme při řešení soustavy regulárních rovnic?

    1.  Pokud v soustavě existuje rovnice pro neznámou $x$, která na pravé straně neznámou $x$ neobsahuje (obsahuje pouze regulární výrazy bez neznámé, či s jinou neznámou), za neznámou $x$ dosadíme do všech zbylých rovnic a danou rovnici zrušíme. Například rovnici $x = 1+y$ bychom zrušili a za neznámou $x$ dosadili do všech zbylých rovnic její pravou stranu, tedy $1+y$.
    2.  Pokud rovnice pro neznámou $x$ obsahuje neznámou $x$ i na pravé straně (např. $x = 1^*x + 2y + 1$), poté tuto rovnici řešíme standardně dle toho, zda jde o levou nebo pravou regulární rovnici.
    3.  Pokud jsou nějaké regulární výrazy (bez neznámé) příliš dlouhé, použijeme **substituci**. Například pro $x = x2 + 10^* + \varepsilon + 00 + 011^*$ lze zavést substituci $A = 10^* + \varepsilon + 00 + 011^*$. Dále tedy pracujeme s rovnicí $x = x2 + A$.
