# 6.1 Neomezené gramatiky

<a id="def-neomezena-gramatika"></a>
!!! Definition "[Definice (Neomezená gramatika)](#def-neomezena-gramatika)"

    ### Neomezená gramatika {#def-neomezena-gramatika}

    **Neomezená gramatika** je uspořádaná čtveřice $G = (N, \Sigma, P, S)$, kde pravidla $P$ jsou tvaru:

    $\alpha A \beta \to \gamma \quad (\alpha, \beta, \gamma \in (N \cup \Sigma)^*, A \in N)$
